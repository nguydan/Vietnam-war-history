Digital Archive Architecture

This project is a bespoke digital humanities archive designed for educational accessibility and high-performance mobile browsing. It avoids heavy frameworks in favor of native web technologies.

Frontend Engine

Built with Vanilla JavaScript. The timeline.js script dynamically fetches and renders data, while toggle.js handles the bilingual state management.

Data Layer

Uses a flat-file JSON database (events.json) to store archival metadata, allowing for easy updates and content expansion.

Visual System

Custom CSS3 featuring a responsive grid layout. The color palette is inspired by military field manuals and archival documents.

Media Hosting

Archival footage is integrated via the YouTube IFrame API, optimized for mobile embedding with custom aspect-ratio containers.
